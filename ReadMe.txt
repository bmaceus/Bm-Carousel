
.bm-carousel
---------------------------------------------------------------------------------
A height for the .bm-carousel container is required for responsiveness. 

bm-slider-timer             =>              set amount of milliseconds to animate each slide. The default eight is 500px
======================================



.bm-carousel-inner
---------------------------------------------------------------------------------
A height for the .bm-carouse-inner container is required for responsiveness.  he default eight is 500px



======================================
.Carousel-item
---------------------------------------------------------------------------------
bm-data-bck-clr             =>              animate(fade) your slide background color



======================================

Each DIRECT (>) child of the ".Carousel-item" container may have the following attributes
---------------------------------------------------------------------------------
bm-data-delay               =>              wait a set amount of milliseconds to animate this item

bm-data-ease-in             =>              how long should it take to animate this item when showing 

bm-data-ease-out            =>              how long should it take to animate this item when hiding 

bm-data-in                  =>              how to animate this item when showing (
                                            fade, fadeLeft, fadeRight, fadeUp, fadeDown
                                            slideLeft, slideRight, slideUp, slideDown
                                            )

bm-data-out                 =>              how to animate this item when hiding (
                                            fade, fadeLeft, fadeRight, fadeUp, fadeDown
                                            slideLeft, slideRight, slideUp, slideDown
                                            )

bm-data-top                 =>              top position of this item in px

bm-data-left                =>              left position of this item in px
